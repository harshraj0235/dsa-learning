Bit Manipulation

Bit Manipulation ek technique hai jisme bits ke level par operations perform kiye jate hain. Yeh operations fast aur efficient hote hain, aur inhe low-level programming tasks ke liye use kiya jata hai.

Example ke liye, neeche ek integer ke bits ko count karne ka code: 

int countSetBits(int n) {
    int count = 0;
    while (n) {
        count += n & 1;
        n >>= 1;
    }
    return count;
}

Bit Manipulation Operations

Bit Manipulation par kuch common operations perform kiye ja sakte hain:

  • AND Operation: Do bits ko AND karna.
  • OR Operation: Do bits ko OR karna.
  • XOR Operation: Do bits ko XOR karna.
  • NOT Operation: Ek bit ko invert karna.
  • Left Shift: Bits ko left shift karna.
  • Right Shift: Bits ko right shift karna.

Example ke liye, ek number ko left shift karne ka code: 

int leftShift(int n, int k) {
    return n << k;
}
