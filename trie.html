Trie

Trie ek tree-like data structure hai jo strings ko efficiently store aur search karne ke liye use hota hai. Trie ko prefix tree bhi kaha jata hai kyunki yeh common prefixes ko share karta hai. Trie ko mainly dictionary words ko store karne, auto-complete suggestions, aur spell-checking ke liye use kiya jata hai.

Trie Operations

Trie par kuch common operations perform kiye ja sakte hain:

  • Insertion: Trie mein ek naya word add karna.
  • Search: Trie mein ek word ko search karna.
  • Deletion: Trie se ek word remove karna.
  • Prefix Search: Trie mein ek prefix ke saath words ko search karna.

Example ke liye, ek Trie ko build karne aur search karne ka code: 

class TrieNode {
    constructor() {
        this.children = {};
        this.isEndOfWord = false;
    }
}

class Trie {
    constructor() {
        this.root = new TrieNode();
    }

    insert(word) {
        let node = this.root;
        for (let char of word) {
            if (!node.children[char]) {
                node.children[char] = new TrieNode();
            }
            node = node.children[char];
        }
        node.isEndOfWord = true;
    }

    search(word) {
        let node = this.root;
        for (let char of word) {
            if (!node.children[char]) {
                return false;
            }
            node = node.children[char];
        }
        return node.isEndOfWord;
    }

    delete(word) {
        this.deleteHelper(this.root, word, 0);
    }

    deleteHelper(node, word, depth) {
        if (!node) {
            return false;
        }

        if (depth === word.length) {
            if (node.isEndOfWord) {
                node.isEndOfWord = false;
            }

            return Object.keys(node.children).length === 0;
        }

        const char = word[depth];
        if (this.deleteHelper(node.children[char], word, depth + 1)) {
            delete node.children[char];
            return Object.keys(node.children).length === 0 && !node.isEndOfWord;
        }

        return false;
    }
}
