Graphs

Graphs ek aise data structure hain jo nodes (vertices) aur edges (connections) se milkar bante hain. Graphs ko various applications jaise social networks, web page ranking, and network routing ke liye use kiya jata hai.

Graphs ko use karne ka reason yeh hai ki yeh complex relationships aur connections ko efficiently represent karte hain. Graphs ko various operations jaise traversal, searching, shortest path finding, etc. ke liye use kiya jata hai.

Graph Operations

Graph par kuch common operations perform kiye ja sakte hain:

  • Traversal: Graph ke nodes ko visit karna (BFS, DFS).
  • Shortest Path: Do nodes ke beech ka shortest path find karna (Dijkstra, Bellman-Ford).
  • Cycle Detection: Graph mein cycle detect karna.
  • Minimum Spanning Tree: Graph ka minimum spanning tree find karna (Kruskal, Prim).
  • Topological Sort: Directed Acyclic Graph (DAG) ka topological order find karna.

Example ke liye, ek graph ko BFS traversal karne ka code: 

class Graph {
    constructor(vertices) {
        this.V = vertices;
        this.adjList = new Map();
    }

    addVertex(v) {
        this.adjList.set(v, []);
    }

    addEdge(v, w) {
        this.adjList.get(v).push(w);
    }

    bfs(startingNode) {
        let visited = new Array(this.V).fill(false);
        let queue = [];
        visited[startingNode] = true;
        queue.push(startingNode);

        while (queue.length !== 0) {
            let getQueueElement = queue.shift();
            console.log(getQueueElement);

            let getList = this.adjList.get(getQueueElement);
            for (let i in getList) {
                let neigh = getList[i];
                if (!visited[neigh]) {
                    visited[neigh] = true;
                    queue.push(neigh);
                }
            }
        }
    }
}
