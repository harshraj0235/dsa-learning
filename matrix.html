Matrix Data Structure

Matrix ek 2D array hota hai jisme rows aur columns hote hain. Yeh ek rectangular array hota hai jisme elements ko rows aur columns ke form mein arrange kiya jata hai.

Example ke liye, neeche matrix ko consider karein: 

int matrix[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

Yaha, matrix matrix mein 3 rows aur 3 columns hain, aur har element ko uske row aur column index se access kiya ja sakta hai.

Matrix Operations

Matrix par kuch common operations perform kiye ja sakte hain:

  • Traversal: Matrix ke har element ko access karna.
  • Insertion: Matrix mein ek naya element add karna.
  • Deletion: Matrix se ek element remove karna.
  • Searching: Matrix mein ek specific element ko dhoondhna.
  • Updating: Matrix ke ek element ko update karna.

Example ke liye, ek matrix ko traverse karne ka code: 

for(int i = 0; i < 3; i++) {
    for(int j = 0; j < 3; j++) {
        printf("%d ", matrix[i][j]);
    }
    printf("\n");
}

Pattern Data Structure

Pattern printing ek common programming task hai jisme specific patterns ko print kiya jata hai using loops. Yeh task logical thinking aur loop control ko improve karta hai.

Example ke liye, neeche ek simple pattern ko consider karein: 

*
* *
* * *
* * * *
* * * * *

Is pattern ko print karne ka code: 

for(int i = 1; i <= 5; i++) {
    for(int j = 1; j <= i; j++) {
        printf("* ");
    }
    printf("\n");
}
