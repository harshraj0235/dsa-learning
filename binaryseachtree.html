Binary Search Tree (BST)

Binary Search Tree (BST) ek binary tree hai jisme har node ke left subtree ke saare nodes ka value us node ke value se chhota hota hai aur right subtree ke saare nodes ka value us node ke value se bada hota hai. BST ko searching, sorting, aur dynamic set operations ke liye use kiya jata hai.

Example ke liye, neeche ek binary search tree ka illustration hai: 

        4
       / \
      2   6
     / \ / \
    1  3 5  7

Binary Search Tree ko use karne ka reason yeh hai ki yeh efficient searching aur sorting operations ko support karta hai aur inhe various applications jaise database indexing, file system, etc. mein use kiya jata hai.

Binary Search Tree Operations

Binary Search Tree par kuch common operations perform kiye ja sakte hain:

  • Insertion: BST mein ek naya node add karna.
  • Deletion: BST se ek node remove karna.
  • Searching: BST mein ek specific node ko dhoondhna.
  • Traversal: BST ke nodes ko visit karna (Inorder, Preorder, Postorder).
  • Find Min/Max: BST mein minimum aur maximum value ko find karna.
  • Find Successor/Predecessor: BST mein ek node ka inorder successor aur predecessor find karna.

Example ke liye, ek binary search tree mein value ko search karne ka code: 

class Node {
    constructor(data) {
        this.data = data;
        this.left = null;
        this.right = null;
    }
}

function searchBST(root, key) {
    if (root == null || root.data == key) return root;
    if (root.data < key) return searchBST(root.right, key);
    return searchBST(root.left, key);
}
