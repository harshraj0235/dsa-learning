String Data Structure

Concept: Strings are sequences of characters, often used for text manipulation. Strings are one of the most commonly used data structures in programming. They are used to store and manipulate text.

Example ke liye, neeche string ko consider karein: 

char str[] = "Hello, World!";

Yaha, string str mein "Hello, World!" text store hai.

String Operations

Strings par kuch common operations perform kiye ja sakte hain:

  • Traversal: String ke har character ko access karna.
  • Concatenation: Do ya zyada strings ko jodna.
  • Substring: String ke ek part ko extract karna.
  • Searching: String mein ek specific character ya substring ko dhoondhna.
  • Comparison: Do strings ko compare karna.
  • Length: String ki length ko find karna.

Example ke liye, ek string ko traverse karne ka code: 

char str[] = "Hello, World!";
for(int i = 0; str[i] != '\0'; i++) {
    printf("%c ", str[i]);
}
