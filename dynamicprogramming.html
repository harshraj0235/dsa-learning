Dynamic Programming

Dynamic Programming (DP) ek algorithmic technique hai jo complex problems ko solve karne ke liye use hoti hai by breaking them down into simpler subproblems. DP ko optimization problems ko solve karne ke liye use kiya jata hai jaha overlapping subproblems aur optimal substructure properties hoti hain.

Example ke liye, neeche ek dynamic programming algorithm ka illustration hai jo local optimal choices ko select kar raha hai: 

Illustration of a dynamic programming algorithm making local optimal choices

Dynamic Programming algorithms ko use karne ka reason yeh hai ki yeh complex problems ko efficiently solve karte hain by storing the results of subproblems to avoid redundant computations.

Dynamic Programming Operations

Dynamic Programming par kuch common operations perform kiye ja sakte hain:

  • Memoization: Recursive approach with caching of subproblem results.
  • Tabulation: Iterative approach with a table to store subproblem results.
  • Optimal Substructure: Problem can be broken down into optimal subproblems.
  • Overlapping Subproblems: Subproblems are solved multiple times.

Example ke liye, ek dynamic programming algorithm ka code jo Fibonacci sequence ko solve karta hai: 

int fib(int n) {
    int f[n+2];
    f[0] = 0;
    f[1] = 1;
    for (int i = 2; i <= n; i++) {
        f[i] = f[i-1] + f[i-2];
    }
    return f[n];
}
