Array Data Structure

Array ek collection hota hai items ka jo contiguous memory locations mein store hote hain. Iska idea yeh hai ki multiple items ko ek saath store kiya jaye jo same type ke ho. Isse har element ka position calculate karna easy ho jata hai by simply adding an offset to a base value, i.e., array ke first element ka memory location (generally array ka naam denote karta hai).

Example ke liye, neeche array ko consider karein: 

int arr[5] = {1, 2, 3, 4, 5};

Yaha, array arr mein 5 elements hain, aur har element ko uske index se access kiya ja sakta hai, jo 0 se 4 tak hota hai.

Array Operations

Arrays par kuch common operations perform kiye ja sakte hain:

  • Traversal: Array ke har element ko access karna.
  • Insertion: Array mein ek naya element add karna.
  • Deletion: Array se ek element remove karna.
  • Searching: Array mein ek specific element ko dhoondhna.
  • Updating: Array ke ek element ko update karna.

Example ke liye, ek array ko traverse karne ka code: 

for(int i = 0; i < 5; i++) {
    printf("%d ", arr[i]);
}
