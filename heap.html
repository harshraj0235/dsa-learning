Heaps

Heaps ek special tree-based data structure hain jo complete binary tree ke form mein hote hain. Heaps ko mainly priority queues ke implementation ke liye use kiya jata hai. Heaps do types ke hote hain: Max Heap aur Min Heap.

Max Heap mein, parent node ka value hamesha child nodes ke values se bada hota hai. Min Heap mein, parent node ka value hamesha child nodes ke values se chhota hota hai.

Heap Operations

Heap par kuch common operations perform kiye ja sakte hain:

  • Insertion: Heap mein ek naya element add karna.
  • Deletion: Heap se root element remove karna.
  • Heapify: Heap property ko maintain karna.
  • Extract Max/Min: Max Heap se maximum aur Min Heap se minimum element ko extract karna.
  • Build Heap: Ek unsorted array se heap banana.

Example ke liye, ek Max Heap ko build karne ka code: 

class MaxHeap {
    constructor() {
        this.heap = [];
    }

    insert(value) {
        this.heap.push(value);
        this.heapifyUp();
    }

    heapifyUp() {
        let index = this.heap.length - 1;
        while (index > 0) {
            let element = this.heap[index];
            let parentIndex = Math.floor((index - 1) / 2);
            let parent = this.heap[parentIndex];

            if (parent >= element) break;

            this.heap[index] = parent;
            this.heap[parentIndex] = element;
            index = parentIndex;
        }
    }

    extractMax() {
        const max = this.heap[0];
        const end = this.heap.pop();
        if (this.heap.length > 0) {
            this.heap[0] = end;
            this.heapifyDown();
        }
        return max;
    }

    heapifyDown() {
        let index = 0;
        const length = this.heap.length;
        const element = this.heap[0];

        while (true) {
            let leftChildIndex = 2 * index + 1;
            let rightChildIndex = 2 * index + 2;
            let leftChild, rightChild;
            let swap = null;

            if (leftChildIndex < length) {
                leftChild = this.heap[leftChildIndex];
                if (leftChild > element) {
                    swap = leftChildIndex;
                }
            }

            if (rightChildIndex < length) {
                rightChild = this.heap[rightChildIndex];
                if (
                    (swap === null && rightChild > element) ||
                    (swap !== null && rightChild > leftChild)
                ) {
                    swap = rightChildIndex;
                }
            }

            if (swap === null) break;

            this.heap[index] = this.heap[swap];
            this.heap[swap] = element;
            index = swap;
        }
    }
}
