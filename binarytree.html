Binary Trees

Binary Trees ek hierarchical data structure hai jisme har node ke maximum do children hote hain, left child aur right child. Binary Trees ko various applications jaise expression parsing, searching, and sorting ke liye use kiya jata hai.

Example ke liye, neeche ek binary tree ka illustration hai: 

        1
       / \
      2   3
     / \ / \
    4  5 6  7

Binary Trees ko use karne ka reason yeh hai ki yeh hierarchical data ko efficiently manage karte hain aur inhe various operations jaise traversal, insertion, deletion, etc. ke liye use kiya jata hai.

Binary Tree Operations

Binary Tree par kuch common operations perform kiye ja sakte hain:

  • Traversal: Tree ke nodes ko visit karna (Inorder, Preorder, Postorder, Level Order).
  • Insertion: Tree mein ek naya node add karna.
  • Deletion: Tree se ek node remove karna.
  • Searching: Tree mein ek specific node ko dhoondhna.
  • Height: Tree ki height ko find karna.
  • Diameter: Tree ka diameter find karna.

Example ke liye, ek binary tree ko inorder traversal karne ka code: 

class Node {
    constructor(data) {
        this.data = data;
        this.left = null;
        this.right = null;
    }
}

function inorderTraversal(node) {
    if (node == null) return;
    inorderTraversal(node.left);
    console.log(node.data + " ");
    inorderTraversal(node.right);
}
