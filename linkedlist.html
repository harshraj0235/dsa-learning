Linked List Data Structure

Linked List ek linear data structure hai jisme elements ko nodes ke form mein store kiya jata hai. Har node mein data aur next node ka reference hota hai. Linked List ko dynamic memory allocation aur efficient insertion/deletion operations ke liye use kiya jata hai.

Example ke liye, neeche ek simple linked list ko consider karein: 

struct Node {
    int data;
    struct Node* next;
};

struct Node* head = NULL;

Yaha, linked list head node se start hoti hai.

Linked List Operations

Linked List par kuch common operations perform kiye ja sakte hain:

  • Traversal: Linked list ke har node ko access karna.
  • Insertion: Linked list mein ek naya node add karna.
  • Deletion: Linked list se ek node remove karna.
  • Searching: Linked list mein ek specific node ko dhoondhna.
  • Reversal: Linked list ke nodes ko reverse order mein arrange karna.

Example ke liye, ek linked list ko traverse karne ka code: 

void printList(struct Node* node) {
    while (node != NULL) {
        printf("%d ", node->data);
        node = node->next;
    }
}
