Find the Maximum and Minimum Element in an Array

Solution Explanation

Chalo samjhte hain kaise find karte hain maximum aur minimum element ek array mein step by step:

  1. Pehle, humein ek array lena hoga jismein humein maximum aur minimum element find karna hai. Example ke liye, hum ek array lete hain [3, 5, 1, 8, 2].
  2. Hum do variables use karenge: ek jo maximum value ko store karega (max) aur ek jo minimum value ko store karega (min).
  3. Hum array ke har element ko iterate karenge aur check karenge agar current element max se bada hai to max ko update karenge aur agar current element min se chhota hai to min ko update karenge.
  4. Aakhir mein, max aur min variables mein maximum aur minimum values store ho jaayengi.

Java Code

Method 1: Using Iteration

Step 1: Pehle ek array define karte hain:


int[] array = {3, 5, 1, 8, 2};

Step 2: MaxMin function ko call karte hain:


int[] result = findMaxMin(array);

Step 3: MaxMin function ko define karte hain:


public static int[] findMaxMin(int[] array) {
    int max = Integer.MIN_VALUE;
    int min = Integer.MAX_VALUE;
    for (int num : array) {
        if (num > max) {
            max = num;
        }
        if (num < min) {
            min = num;
        }
    }
    return new int[]{max, min};
}

Step 4: Result ko print karte hain:


System.out.println("Max: " + result[0] + ", Min: " + result[1]);

Method 2: Using Recursion

Step 1: Pehle ek array define karte hain:


int[] array = {3, 5, 1, 8, 2};

Step 2: MaxMin function ko call karte hain:


int[] result = findMaxMin(array, 0, array.length - 1);

Step 3: MaxMin function ko define karte hain:


public static int[] findMaxMin(int[] array, int start, int end) {
    if (start == end) {
        return new int[]{array[start], array[start]};
    }
    if (end == start + 1) {
        if (array[start] > array[end]) {
            return new int[]{array[start], array[end]};
        } else {
            return new int[]{array[end], array[start]};
        }
    }
    int mid = (start + end) / 2;
    int[] left = findMaxMin(array, start, mid);
    int[] right = findMaxMin(array, mid + 1, end);
    return new int[]{Math.max(left[0], right[0]), Math.min(left[1], right[1])};
}

Step 4: Result ko print karte hain:


System.out.println("Max: " + result[0] + ", Min: " + result[1]);

def find_max_min(arr):
    max_val = float('-inf')
    min_val = float('inf')
    for num in arr:
        if num > max_val:
            max_val = num;
        if num < min_val:
            min_val = num;
    return max_val, min_val

    if start == end:

    if end == start + 1:
        if arr[start] > arr[end]:
            return arr[start], arr[end]

C++ Code

Method 1: Using Iteration

Step 1: Pehle ek array define karte hain:


int arr[] = {3, 5, 1, 8, 2};

Step 2: MaxMin function ko define karte hain:


pair findMaxMin(int arr[], int n) {
    int max_val = INT_MIN;
    int min_val = INT_MAX;
    for (int i = 0; i < n; i++) {
        if (arr[i] > max_val) {
            max_val = arr[i];
        }
        if (arr[i] < min_val) {
            min_val = arr[i];
        }
    }
    return {max_val, min_val};
}

Step 3: MaxMin function ko call karte hain:


pair result = findMaxMin(arr, sizeof(arr) / sizeof(arr[0]));

Step 4: Result ko print karte hain:


cout << "Max: " << result.first << ", Min: " << result.second << endl;

Method 2: Using Recursion

Step 1: Pehle ek array define karte hain:


int arr[] = {3, 5, 1, 8, 2};

Step 2: MaxMin function ko define karte hain:


pair findMaxMin(int arr[], int start, int end) {
    if (start == end) {
        return {arr[start], arr[start]};
    }
    if (end == start + 1) {
        if (arr[start] > arr[end]) {
            return {arr[start], arr[end]};
        } else {
            return {arr[end], arr[start]};
        }
    }
    int mid = (start + end) / 2;
    pair left = findMaxMin(arr, start, mid);
    pair right = findMaxMin(arr, mid + 1, end);
    return {max(left.first, right.first), min(left.second, right.second)};
}
