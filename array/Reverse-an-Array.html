Reverse an Array/String

Solution Explanation

Chalo samjhte hain kaise reverse karte hain ek array ya string ko step by step:

  1. Pehle, humein ek array ya string lena hoga jo humein reverse karna hai. Example ke liye, hum ek array lete hain [1, 2, 3, 4, 5].
  2. Hum do pointers use karenge: ek jo shuru se start hoga (start) aur ek jo end se start hoga (end) array ke.
  3. Hum elements ko swap karenge jo start aur end pointers pe hain.
  4. Swap karne ke baad, hum start pointer ko ek step aage badhaenge aur end pointer ko ek step peeche le jaayenge.
  5. Hum swapping process ko repeat karenge jab tak start pointer end pointer se chhota hai.
  6. Aakhir mein, array reverse ho jaayega. Example ke liye, [1, 2, 3, 4, 5] ban jaayega [5, 4, 3, 2, 1].

Java Code

Method 1: Using Two Pointers

Step 1: Pehle ek array define karte hain:


int[] array = {1, 2, 3, 4, 5};

Step 2: Reverse function ko call karte hain:


reverse(array);

Step 3: Reverse function ko define karte hain:


public static void reverse(int[] array) {
    int start = 0;
    int end = array.length - 1;
    while (start < end) {
        int temp = array[start];
        array[start] = array[end];
        array[end] = temp;
        start++;
        end--;
    }
}

Step 4: Reversed array ko print karte hain:


for (int i : array) {
    System.out.print(i + " ");
}

Method 2: Using Recursion

Step 1: Pehle ek array define karte hain:


int[] array = {1, 2, 3, 4, 5};

Step 2: Reverse function ko call karte hain:


reverse(array, 0, array.length - 1);

Step 3: Reverse function ko define karte hain:


public static void reverse(int[] array, int start, int end) {
    if (start >= end) {
        return;
    }
    int temp = array[start];
    array[start] = array[end];
    array[end] = temp;
    reverse(array, start + 1, end - 1);
}

Step 4: Reversed array ko print karte hain:


for (int i : array) {
    System.out.print(i + " ");
}

Method 3: Using Collections.reverse()

Step 1: Pehle ek array define karte hain:


Integer[] array = {1, 2, 3, 4, 5};

Step 2: Collections.reverse() function ko call karte hain:


Collections.reverse(Arrays.asList(array));

Step 3: Reversed array ko print karte hain:

}

Python Code

Method 1: Using Two Pointers

Step 1: Pehle ek array define karte hain:


array = [1, 2, 3, 4, 5]

Step 2: Reverse function ko define karte hain:


def reverse_array(arr):
    start = 0
    end = len(arr) - 1
    while start < end:
        arr[start], arr[end] = arr[end], arr[start]
        start += 1
        end -= 1
    return arr

Step 3: Reverse function ko call karte hain aur result ko print karte hain:


print(reverse_array(array))

Method 2: Using Recursion

Step 1: Pehle ek array define karte hain:


array = [1, 2, 3, 4, 5]

Step 2: Reverse function ko define karte hain:


def reverse_array(arr, start, end):

    reverse_array(arr, start + 1, end - 1)

reverse_array(array, 0, len(array) - 1)
print(array)

Method 3: Using Slicing

Step 1: Pehle ek array define karte hain:


array = [1, 2, 3, 4, 5]

Step 2: Slicing ka use karke array ko reverse karte hain:


array = array[::-1]

Step 3: Reversed array ko print karte hain:


print(array)
